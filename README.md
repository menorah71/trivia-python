# Juego de Trivia - Python

Juego interactivo de preguntas y respuestas sobre tecnologia y Python.
Las preguntas se mezclan aleatoriamente en cada partida.

## Como funciona?

1. El jugador escribe su nombre
2. El programa carga y mezcla las preguntas del archivo JSON
3. Se muestran 5 preguntas con 4 opciones cada una
4. Al final se muestra el puntaje y nivel obtenido

## Niveles

- 50/50 -> Maestro Python!
- 40/50 -> Experto!
- 30/50 -> Avanzado!
- 20/50 -> En progreso!
- menos de 20 -> Sigue estudiando!

## Tecnologias

- Python 3
- Libreria json (incluida en Python)
- Libreria random (incluida en Python)

## Estructura del proyecto

- trivia.ipynb   -> el juego completo
- preguntas.json -> banco de 10 preguntas

## Como usarlo?

1. Clona el repositorio
2. Abre trivia.ipynb en Jupyter Notebook
3. Ejecuta todas las celdas en orden
4. Escribe tu nombre y responde las preguntas
